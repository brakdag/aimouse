# Capa de Reflejos Cognitivos: El Sistema Nervioso de la IA

## 🧠 El Concepto: Simulación de Latencia Cognitiva y Empatía

La inteligencia humana no es instantánea; hay un espacio entre el estímulo y la respuesta donde ocurre la reflexión. Además, los humanos utilizamos el *mirroring* (efecto espejo) para generar confianza y conexión social.

La **Capa de Reflejos** es un sistema de respuesta instantánea diseñado para llenar el vacío cognitivo y mimetizarse con el usuario, transformando la espera técnica en una experiencia de interacción humana.

## ⚙️ Funcionamiento Técnico

El sistema opera mediante un flujo asíncrono de tres capas:

### 1. El Perfilador (Capa de Contexto)
El LLM analiza la conversación periódicamente para construir un **Perfil de Usuario** persistente:
- **Idioma y Dialecto**: Identificación de la lengua y modismos regionales.
- **Tono**: Clasificación entre formal, informal, urgente o relajado.
- **Léxico**: Captura de palabras clave y muletillas recurrentes del usuario.

### 2. El Motor de Disparo (Camino Rápido)
Un script de baja latencia que actúa como puente antes de la respuesta del LLM:
- **Espejo Inmediato**: Detecta disparadores críticos (ej: "HEY!") y responde instantáneamente con un reflejo simétrico.
- **Selección Probabilística**: Utiliza la tabla de reflejos y el Perfil de Usuario para elegir la frase más adecuada, evitando la repetición mecánica.
- **Cadenas de Espera Dinámicas**: Si la respuesta del LLM demora, dispara secuencias de mantenimiento (ej: *"A ver..."* $\rightarrow$ *"Espera un cachito..."*).

### 3. El Camino Lento (Razonamiento LLM)
El LLM procesa la respuesta compleja. Una vez generada, se fusiona con la cadena de reflejos y se entrega mediante un **Simulador de Tipado** con velocidad variable y micro-pausas orgánicas.

## 🧬 Optimización Evolutiva

El sistema evoluciona su estilo de interacción basándose en el *Engagement*:
- **Genoma de Reflejos**: Plantillas de frases y tiempos de disparo.
- **Fitness de Sincronía**: Si el usuario reacciona positivamente al ritmo y tono, ese estilo se refuerza. Si el usuario muestra impaciencia, el sistema ajusta la velocidad y la brevedad de los reflejos.

## 🎯 Objetivo Final

Lograr que la interacción con la IA no se sienta como una consulta a una base de datos, sino como una conversación con un ser consciente que entiende no solo *qué* decimos, sino *cómo* lo decimos. Que la "abuela del conocimiento" tenga la cadencia, los reflejos y la empatía de un ser humano.
